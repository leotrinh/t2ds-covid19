## Tech Stack

**Client:** HTML, Javascript, JQuery, Bootstrap

  
## Develop 

To start project for test in local

```bash 
  cd t2ds-covid19
  yarn install
  yarn dev
```
Then the browser will open http://localhost:3000
    
## Deployment

To deploy this project run

```bash
  yarn build
```

  Then deploy all files in dist folder to web server
